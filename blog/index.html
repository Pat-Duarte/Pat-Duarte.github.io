Error message

Overview Explain the reason for the error in plain language. Provide only relevant and useful information. Give users a way out. Before We can’t share this for you The file ‘filename’ is too large to be shared to our servers. To make shares load faster, we restrict the size of each uploaded file/image to 6MB. Can you make the file smaller and try again? Another option is exporting to HTML with no size limit.

Title

Overview Keep titles goal-oriented and relevant to the purpose of the topic. Before Getting started This title is too generic. When reading it, the user can’t have an idea of what the topic will be about. In the case of a tutorial, they don’t know what they will achieve after following the instructions. Too vague. What’s the purpose? After Creating your first artboard This title reflects a concrete action and it gives the user an idea of what the goal of the tutorial is.
