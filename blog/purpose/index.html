Purpose

Overview

Make the purpose of the document clear. Give each document a single purpose.

Before

Multiple purposes. Long sentence.

This topic describes the process of downloading and adding the Indigo Design Libraries into Sketch and how to craft your first design using and customizing the Components necessary for building a registration screen.

After

Single purpose with simple language.

At the end of this topic you’ll have a Sketch Artboard for a registration screen using Indigo Design Libraries.
