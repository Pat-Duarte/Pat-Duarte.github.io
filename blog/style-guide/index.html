Grammar

  • Write in the present simple
  • Avoid passive voice
  • If a sentence is too long, split it

Vocabulary

US spelling Avoid adverbs Does it add value? How many do I need to put my meaning across? Keep it simple in order to → to as well as → and disseminate → distribute advancement → improvement utilize → use Don’t overuse common words Ask yourself these questions: Does it add value? Is the meaning clear even if I leave it out? Is there a simpler way of expressing this? Think twice before using these: Ensure Also Leverage Avoid “etc.” Never use it at the end these series, because they already imply there are other examples.: for example e.g. including such as Whether Don’t write “or not” at the end of a clause including “whether”. It’s implied.

Punctuation Favor simple “or” instead of “and/or” Slash Favor “or” Don’t leave spaces before or after the slash in a sentence Use the Oxford comma The Oxford (or serial) comma is the final comma in a list of things.
